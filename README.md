### 启动方式

> 因为浏览器的限制,不能再通过直接打开 index.html 的方式运行

如果是用 vscode 打开的，可以下一个 `Live Server`

如果你用的 python，可以在本项目根目录下，快速启动一个 http 服务

```
 python -m http.server 5500
```

然后访问 <a>127.0.0.1:5500/index.html</a>
